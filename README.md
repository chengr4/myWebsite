# Build a Responsive Website

## Using tech
### font awesome
1. Go to [cdnjs.com](https://cdnjs.com/)
2. Search font and go into font-awesome
3. Copy link tag with *all.min.css* to index.html
4. Then font-awesome classes can be used to create icons
